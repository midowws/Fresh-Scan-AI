Tentang Proyek

Informasi mengenai latar belakang, teknologi, dan tim pengembang.

🎯

Tujuan Proyek

FreshScan AI dikembangkan untuk membantu konsumen maupun pelaku usaha dalam melakukan penilaian kualitas bahan pangan secara cepat, mudah, dan objektif, tanpa memerlukan keahlian khusus dalam menilai kesegaran buah dan sayuran secara manual.

🛠️

Teknologi yang Digunakan

  • Frontend: HTML, CSS, JavaScript
  • Computer Vision: Pengolahan dan ekstraksi fitur gambar
  • Deep Learning: Convolutional Neural Network (CNN)
  • Framework Model: TensorFlow / Keras
  • Bahasa Pemrograman Model: Python
📊

Dataset / Sumber Data

Model dilatih menggunakan kumpulan gambar buah dan sayuran segar maupun tidak segar yang dikumpulkan dari sumber dataset publik (misalnya Kaggle - Fruits Fresh and Rotten for Classification) serta dokumentasi tambahan untuk memperkaya variasi data.

🧠

Model yang Digunakan

Sistem menggunakan arsitektur Convolutional Neural Network (CNN) untuk mengekstraksi fitur visual dari gambar (warna, tekstur, bentuk) dan mengklasifikasikan tingkat kesegaran produk ke dalam kategori Segar atau Tidak Segar beserta skor kepercayaan (confidence score).

Alur Pengembangan Sistem

1

Pengumpulan Data

Mengumpulkan dataset gambar buah dan sayur segar maupun tidak segar dari sumber publik.

2

Pra-pemrosesan

Resize, normalisasi, dan augmentasi gambar agar model lebih general dan robust.

3

Pelatihan Model

Melatih model CNN untuk mengenali pola visual khas bahan pangan segar dan tidak segar.

4

Integrasi Web

Menghubungkan model dengan antarmuka web agar pengguna dapat melakukan prediksi secara langsung.
